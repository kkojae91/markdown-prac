# 강조(Emphasis)

이텔릭체 : 언더바로 감싸준다. _이텔릭_  
두껍게 : 에스터리스크를 두번 넣고 감싸준다. **두껍게**  
이텔릭+두껍게 : **_이텔릭 + 두껍게_**  
취소선 : 물결표시(틸드) 두번 넣고 감싸준다. ~~취소선~~  
밑줄 : u tag로 감사준다. <u>밑줄</u>
